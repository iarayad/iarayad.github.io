# Contents

This folder contains all the content files for the Quarto website:

- **`.qmd` files**: All Quarto markdown files for pages and posts
- **`images/`**: All image assets used in the website
- **`data/`**: Data files used by the website (currently empty)

## Structure

```text
contents/
├── index.qmd           # Main homepage
├── about.qmd          # About page
├── images/            # Image assets
│   ├── logo.svg
│   └── profile.png
└── data/              # Data files
```
